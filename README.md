# Virtual AI Assistant

## Project Description
This project is a basic Virtual AI Assistant chatbot that responds to user queries such as interview tips, SQL questions, resume guidance, and general questions.

## Features
- User can enter questions
- Chatbot gives responses
- Simple web interface
- Built with frontend and backend integration

## Tech Stack
- HTML
- CSS
- JavaScript
- Python
- Flask

## How to Run
1. Open terminal in backend folder
2. Run:
   python app.py

3. Open frontend folder
4. Open index.html in browser

## Example Questions
- What is SQL?
- Give interview tips
- Help with resume
